FileServer test suite is designed to test implementations of file sharing protocol family including:
  • [MS-SMB2] Server Message Block (SMB) Protocol Version 2 and 3, v20190923
  • [MS-FSRVP] File Server Remote VSS Protocol, v20180912
  • [MS-SWN] Service Witness Protocol, v20180912
  • [MS-DFSC] The Distributed File System (DFS): Referral Protocol, v20180912
  • [MS-RSVD] Remote Shared Virtual Disk Protocol, v20180912
  • [MS-FSA] File System Algorithms Protocol, v20190530
  • [MS-SQOS] Storage Quality of Service Protocol, v20180912
  • [MS-HVRS] Hyper-V Remote Storage Profile, v20180912
