Kerberos test suite is designed to test implementations of Kerberos Family Protocols, as specified in the Microsoft documents and their referenced dependencies, including:
  • [MS-KILE] Kerberos Protocol Extensions , v20140212
  • [RFC4120] The Kerberos Network Authentication Service (V5), July 2005
  • [RFC6113] A Generalized Framework for Kerberos Pre-Authentication, April 2011
  • [Referrals-11] Kerberos Principal Name Canonicalization and KDC-Generated Cross-Realm Referrals, July 14, 2008
  • [MS-PAC] Privilege Attribute Certificate Data Structure , v20140212
  • [MS-KKDCP] Kerberos Key Distribution Center (KDC) Proxy Protocol , v20140212
  • [RFC3244] Microsoft Windows 2000 Kerberos Change Password and Set Password Protocols, February 2002
